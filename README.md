# ☕ Manajemen Coffe

Aplikasi web sederhana untuk mengelola sistem kedai kopi seperti data menu, transaksi penjualan, kasir, dan pengelolaan pengguna. Dibuat menggunakan PHP dan MySQL dengan tampilan sederhana serta mudah digunakan.

Repository GitHub: [https://github.com/ekaretno1505-netizen/Manajemen-Coffe](https://github.com/ekaretno1505-netizen/Manajemen-Coffe)

## 👩‍💻 Developer

Eka Retno Rudiati Putri

## 🚀 Fitur

* Login Admin dan Kasir
* Dashboard Sistem
* Data User
* Data Menu Kopi
* Data Kategori Menu
* Transaksi Penjualan
* Sistem Kasir
* Perhitungan Total Pembayaran
* Riwayat Transaksi
* CRUD Data Menu dan Transaksi

## 🛠️ Teknologi

* PHP Native
* MySQL
* HTML
* CSS
* Bootstrap
* JavaScript

## ⚙️ Cara Menjalankan

1. Download atau clone repository ini

2. Pindahkan folder project ke folder:

`htdocs`

3. Import database `.sql` ke phpMyAdmin

4. Jalankan Apache dan MySQL di XAMPP

5. Buka di browser:

`http://localhost/Manajemen-Coffe`

## 📌 Tujuan Project

Project ini dibuat untuk pembelajaran dan latihan membuat aplikasi manajemen kedai kopi berbasis web menggunakan PHP dan MySQL. Sistem ini membantu proses pengelolaan menu, transaksi, dan aktivitas kasir secara digital.
